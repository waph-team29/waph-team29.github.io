Web Application Programming and Hacking

Front-end Web Development Individual Project

Instructor: Dr. Phu Phung

Overview of this course:

This course offers a well-balanced combination of theory and practical experience, providing a thorough investigation of web application development and security. The course will cover the fundamentals of front-end development, including HTML5, JavaScript, and CSS, as well as back-end skills using PHP and MySQL. Understanding the complex architecture of web applications and how client-side and server-side components interact is emphasized. Through hands-on exercises, students will also learn about common web application security flaws and attack strategies. The main objective is to give people the knowledge and abilities to create online applications that are safe, dependable, and capable of detecting, averting, and reducing security threats.
