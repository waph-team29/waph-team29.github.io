Web Application Programming and Hacking

Front-end Web Development Lab

Instructor: Dr. Phu Phung

Overview of this Course:

Course Overview: This course offers a balanced blend of theory and practical experience, providing a comprehensive exploration of web application development and security. Covering the fundamentals of front-end development (HTML5, JavaScript, CSS) and back-end skills (PHP, MySQL), emphasis is placed on understanding the intricate architecture of web applications. Through hands-on exercises, students learn about common security flaws and attack strategies, aiming to equip them with the skills to create secure online applications.
